### Running it locally

#### Requirements

- [Node.js](https://nodejs.org/) (latest)
- [Expo](https://expo.io/) (latest)
- [Yarn](https://yarnpkg.com/)

#### How to run

- `git clone git@github.com:murillo94/cine-the-guide-movie.git`
- `yarn install`

To open the project, choose one:

- `yarn ios`
- `yarn android`
- `yarn start`
